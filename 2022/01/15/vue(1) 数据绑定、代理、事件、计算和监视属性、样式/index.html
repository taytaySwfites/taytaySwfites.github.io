vue是什么？

一套用于构建用户界面的渐进式（自底向上逐层的应用，简单应用 -> 复杂应用）Javascript框架

vue的特点？
1.采用组件化，提高代码复用率、且让代码更好维护
2.声明式编码，无需直接操作DOM，提高开发效率（对应的是命令式编码）
3.虚拟式DOM+Diff算法，尽量复用DOM节点

① .prevent // 阻止默认事件 e.preventDefault()
② .stop // 阻止事件冒泡 e.stopPropogation()

④ .capture // 使用事件捕获 先捕获再冒泡
⑤ .self // 只有event.target是当前操作的元素才触发事件

监视属性：

1.当被监视的属性变化时，回调函数自动调用，进行相关操作
2.监视的属性必须存在，才能进行监视
3.监视分为两种：watch配置 vm.$watch监视

深度监视：

computed和watch的区别：

1.computed能完成的功能，watch也能
2.watch能进行异步任务，而computed不能
3.所有被Vue管理的函数，最好写成普通函数，指向的是 vm 或 组件实例对象
4.所有不被Vue管理的函数，最好写成箭头函数，指向的是 vm 或 组件实例对象（定时器的回调函数、ajax的回调函数，Promise的回调函数）
