CSS3动画animation

2022.07.05 CSS

1.动画

    动画是CSS3中具有颠覆性的特征之一，可以设置多个节点来精确控制一个或一组动画常用来实现复杂的动画效果

2.基本使用

    定义动画@keyframes ——> 使用（调用）

3.定义

    0%（from）是动画的开始，100%（to）是动画的完成（动画序列）

4.常用属性：

属性 描述
@keyframes 规定动画
animation 所有动画属性的简写属性，除了animation-play-state属性
animation-name 指定动画
animation-duration 动画完成一个周期所花费的秒或毫秒
animation-tingming-function 动画的速度曲线
animation-delay 动画何时开始，默认0
animation-iteration-count 动画播放次数，默认1，infinite无限次数
animation-direction 动画是否在下一周期你想播放，默认normal，alternate逆播放
animation-play-state 动画是否正在运行或暂停，默认running，paused暂停
animation-fill-mode 动画结束后状态，保持状态forwards，回到起始backwards

注意：
    ① 暂停动画： animation-play-state: pulsed（经常和鼠标经过等其他配合使用）
    ② 想要动画走回来，而不是直接跳回来： animation-direction: alternate
    ③ 盒子动画结束后，停在结東位置： animation-fill-mode: forwards

5.animation-timing-function

    动画的速度曲线，默认ease

属性 描述
linear 匀速
ease 先慢，再加快，再慢
ease-in 加速
ease-out 减速
ease-in-out 低俗开始和结束
steps 时间函数中的间隔数量（步长）

    steps理解为动画从头到尾，需要多少步完成

打字机效果：
1ch = 一个英文 = 一个数字 2ch = 1个中文

6.animation-direction

    动画是否逆向播放
    ① normal：默认的
    ② reverse：从终点运动向起点（终点=>起点）
    ③ alternate：到达终点后是否原路返回（起点=>终点=>起点）
    ④ alternate-reverse：（终点=>起点=>终点）animation-iteration-count < 2时无效

7.animation-fill-mode

    动画结束后状态
    ① forwards：保持当前位置
    ② backwards：回到初始位置
