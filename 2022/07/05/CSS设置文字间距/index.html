CSS 设置文字间距

2022.07.05 CSS

1.word-spacing

    设置字间距（单词的间距），“字”定义为由空白字符包围的一个字符串（该属性以空格为基准进行调节间距的；如果多个字母连在一起，则会被word-spacing设为一个单词；如果汉字被空格分隔，则分隔的多个汉字就被设为不同的单词）<br/?

语法：
    ① normal：定义单词间的标准空间，默认值
    ② length：定义单词间的固定长度（长度值）
    ③ inherit：规定从父元素继承word-spacing属性的值

注意：
    如果指定为length，会调整字之间的通常间隔，所以nromal就等同于0，允许福长度的值，这样会让字之间挤得更紧


2.letter-spacing

    增加或减少字符间的空白（字符间距），该属性定义了在文本字符框之间插入多少空间。由于字符字型通常比其字符框要窄，指定长度值时，会调整字母之间通常的间隔

注意：
    一个中文文字、一个英文字母作为一个“字”


3.word-spacing和letter-spacing的比较

word-spacing letter-spacing
检索 对象的单词之间插入的空格 每个文字
长度值 允许允许负数 允许允许负数
中英文 适用于英语使用 中英文都适用
其他 对汉字，空格分隔才有效 只对文字起作用 对于图片失效的
