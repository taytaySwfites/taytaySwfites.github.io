CSS 自定义属性

2022.04.25 CSS

使用CSS自定义属性（变量） - CSS（层叠样式表） | MDN (mozilla.org)

自定义属性：CSS变量、级联变量，可以重复使用的

基本用法：属性名以--开始，属性值可以是任何有效的CSS值

注意：规则集指定的选择器定义的自定义属性的可见作用域
最佳是在根伪类:root下，这样在任何地方都能访问到了

注意：自定义属性名是大小写敏感的，–my-color 和 –My-color 会被认为是两个不同的自定义属性

使用：

备用值：当给定值未定义是会用备用纸替换

有效性和值：通过var()函数调用时，在特定上下文环境下也可能不会奏效。属性和自定义变量会导致无效的CSS语句

无效变量会导致什么？
当浏览器遇到无效的 var() 时，会使用继承值或初始值代替

javascript中的值：在javascript中获取或者修改CSS变量和操作普通CSS属性是一样的

在小程序中使用vant，并定制全局主题样式：
在app.wxss中，写入CSS变量，即可对全局生效
